# Java Example

## Prerequisites

- JDK 1.8 or higher
- Maven

## Build and Run

```bash
cd whale

# Set environment variables
export LONGPORT_APP_KEY="your_app_key"
export LONGPORT_APP_SECRET="your_app_secret"
export LONGPORT_ACCESS_TOKEN="your_access_token"

# Run with Maven
mvn clean compile exec:java
```
